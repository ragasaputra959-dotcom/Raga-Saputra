# SIAKAD Mobile App

Aplikasi Sistem Informasi Akademik yang dirancang untuk memudahkan civitas akademika dalam mengakses data akademik.

## Deskripsi
Proyek ini terdiri dari aplikasi mobile Android (client) dan sistem API (back-end) untuk pengelolaan data akademik.

## Teknologi
- Bahasa: Kotlin
- Build System: Gradle (Kotlin DSL)
- ... (tambahkan teknologi lainnya)

## Fitur
- Login Mahasiswa/Dosen
- Akses jadwal kuliah
- ... (tambahkan fitur Anda)

## Cara Menjalankan
1. Clone repositori ini.
2. Buka folder proyek di Android Studio.
3. Pastikan SDK Android sudah terkonfigurasi.
4. Build dan jalankan aplikasi.
